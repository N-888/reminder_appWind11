## 📄 Описание каждого файла

### 🔴 КРИТИЧНЫЕ (Обязательны для работы)

#### `main.py`
- **Тип:** Python скрипт (точка входа)
- **Размер:** ~50 строк
- **Назначение:** Запуск приложения
- **Содержит:** Инициализация всех компонентов
- **Запуск:** `python main.py`
- **Зависит от:** database.py, notifications.py, gui.py

#### `database.py`
- **Тип:** Python модуль (ядро БД)
- **Размер:** ~200 строк
- **Класс:** `ReminderDatabase`
- **Назначение:** SQLite3 операции с напоминаниями
- **Методы:** 20+ для работы с БД
- **Зависит от:** sqlite3 (встроенная библиотека)

#### `notifications.py`
- **Тип:** Python модуль (уведомления)
- **Размер:** ~150 строк
- **Класс:** `NotificationManager`
- **Назначение:** Система уведомлений в реальном времени
- **Методы:** Мониторинг, уведомления, тестирование
- **Зависит от:** database.py, tkinter, win10toast (опционально)

#### `gui.py`
- **Тип:** Python модуль (интерфейс)
- **Размер:** ~400 строк
- **Класс:** `ReminderGUI`
- **Назначение:** Tkinter графический интерфейс
- **Методы:** 20+ для управления окном и напоминаниями
- **Зависит от:** database.py, notifications.py, tkinter

---

### 🟡 ВАЖНЫЕ (Нужны для корректной работы)

#### `config.py`
- **Тип:** Python модуль (конфигурация)
- **Размер:** ~60 строк
- **Назначение:** Константы и настройки приложения
- **Содержит:** Пути, цвета, форматы, интервалы
- **Использование:** Импортируется другими модулями
- **Зависит от:** ничего

#### `requirements.txt`
- **Тип:** Текстовый файл (зависимости)
- **Размер:** 1 строка
- **Содержит:** `win10toast==0.9`
- **Назначение:** Список Python пакетов для установки
- **Команда:** `pip install -r requirements.txt`

#### `run.bat`
- **Тип:** Пакетный файл (batch script)
- **Размер:** ~30 строк
- **Назначение:** Автоматический запуск приложения на Windows
- **Функции:** 
  - Проверка Python
  - Установка зависимостей
  - Запуск main.py
- **Использование:** Двойной клик на файл

### 💾 БАЗА ДАННЫХ

#### `reminders.db`
- **Тип:** SQLite3 база данных
- **Создание:** Автоматически при первом запуске
- **Расположение:** В папке приложения
- **Таблица:** `reminders` с 6 колонками
  - `id` - уникальный идентификатор
  - `title` - заголовок напоминания
  - `description` - описание
  - `due_time` - дата и время срабатывания
  - `status` - текущий статус
  - `created_at` - время создания
- **Размер:** Растёт с добавлением напоминаний (обычно <1 МБ)
