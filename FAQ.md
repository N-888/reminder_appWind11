# ❓ FAQ - Часто задаваемые вопросы

## Установка и запуск

### Q: Как установить приложение?
**A:** Нет установки! Просто выполните:
```bash
pip install -r requirements.txt
python main.py
```

Или используйте `run.bat` на Windows (двойной клик).

---

### Q: Python не установлен, что делать?
**A:** Установите Python 3.7+ с [python.org](https://www.python.org/downloads/)

При установке обязательно отметьте "Add Python to PATH".

---

### Q: Ошибка "ModuleNotFoundError: No module named 'win10toast'"
**A:** Установите зависимости:
```bash
pip install -r requirements.txt
```

---

### Q: Можно ли запустить на Mac или Linux?
**A:** Большинство функций будет работать, но:
- `winsound` не работает (нужен `playsound`)
- `win10toast` не работает (нужны уведомления ОС)

Для Linux установите: `pip install playsound`

## Функционал

### Q: Как создать напоминание?
**A:** 
1. Нажмите **"➕ Добавить"**
2. Введите заголовок и описание
3. Установите дату/время в формате `YYYY-MM-DD HH:MM:SS`
4. Нажмите **"💾 Сохранить"**

Или используйте кнопку быстрого напоминания: **"5 мин"**, **"15 мин"** и т.д.

---

### Q: Какой формат даты?
**A:** Используйте: `YYYY-MM-DD HH:MM:SS`

Примеры:
- `2026-06-06 14:30:00` - 6 июня 2026, 14:30
- `2026-12-25 08:00:00` - Рождество

---

### Q: Как удалить напоминание?
**A:**
1. Выберите напоминание в списке
2. Нажмите **"🗑️ Удалить"**
3. Подтвердите удаление

---

### Q: Как отметить напоминание как выполненное?
**A:**
1. Выберите напоминание
2. Нажмите **"✅ Готово"**

---

### Q: Можно ли отменить напоминание?
**A:** Да! Выберите и нажмите **"❌ Отменить"**

---

### Q: Что означают статусы?
**A:**
- **⏳ Ожидает** - ещё не срабатило
- **✅ Готово** - выполнено
- **⏰ Просрочено** - время прошло, не выполнено
- **❌ Отменено** - отменено пользователем

---

## Уведомления

**A:** Проверьте:
1. Приложение запущено?
2. Время напоминания правильное?
3. Windows разрешает уведомления?

Приложение покажет `pop-up` если системные уведомления недоступны.

---

### Q: Работают ли уведомления при свёрнутом окне?
**A:** Да! Приложение работает в фоне и показывает уведомления независимо от состояния окна.

### Q: Как протестировать уведомления?
**A:** Нажмите кнопку **"🧪 Тест"** - сразу появится тестовое уведомление.

## База данных

### Q: Где хранится база данных?
**A:** В папке приложения: `reminders.db`

Это SQLite3 файл, который создаётся автоматически.

---

### Q: Можно ли перенести БД на другой компьютер?
**A:** Да! Просто скопируйте файл `reminders.db` и положите в папку приложения на новом компьютере.

---

### Q: Как очистить все напоминания?
**A:** Удалите файл `reminders.db` - он создастся заново при следующем запуске.

**Внимание:** Все напоминания будут удалены безвозвратно!

---

### Q: Как экспортировать напоминания?
**A:** В файле `EXAMPLES.md` есть пример экспорта в CSV:
```python
import csv
from database import ReminderDatabase

db = ReminderDatabase()
reminders = db.get_all_reminders()

with open("reminders.csv", "w", newline="") as f:
    writer = csv.DictWriter(f, fieldnames=reminders[0].keys())
    writer.writeheader()
    writer.writerows(reminders)
```

---

### Q: Может ли БД повредиться?
**A:** Маловероятно, но если это произойдёт:
1. Закройте приложение
2. Удалите `reminders.db`
3. Перезапустите приложение
4. БД создастся заново

## Проблемы

### Q: Приложение не запускается
**A:** 
1. Проверьте Python: `python --version`
2. Установите зависимости: `pip install -r requirements.txt`
3. Запустите из терминала: `python main.py`
4. Посмотрите сообщение об ошибке

---

### Q: Кнопки не работают
**A:**
1. Убедитесь, что выбрано напоминание
2. Перезагрузите приложение
3. Проверьте консоль на ошибки

---

### Q: Напоминания не срабатывают в нужное время
**A:**
1. Проверьте системное время Windows
2. Переформатируйте время: `YYYY-MM-DD HH:MM:SS`
3. Используйте кнопку **"🔄 Обновить"** в приложении

---

### Q: При двойном клике ничего не происходит
**A:** 
1. Убедитесь, что клик по напоминанию в списке
2. Попробуйте снова
3. Перезагрузите приложение

---

### Q: БД повреждена, как восстановить?
**A:** К сожалению, не получится восстановить отдельные напоминания. Решение:
1. Удалите `reminders.db`
2. Перезапустите приложение
3. Создавайте напоминания заново

**Совет:** Периодически делайте резервные копии `reminders.db`
