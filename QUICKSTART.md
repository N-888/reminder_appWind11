# 🚀 Быстрый старт - Напоминалка

## Первый запуск

### Шаг 1: Установка Python
Убедитесь, что у вас установлен Python 3.7 или выше:
```bash
python --version
```

### Шаг 2: Установка зависимостей
Откройте командную строку (CMD) в папке проекта и выполните:
```bash
pip install -r requirements.txt
```

### Шаг 3: Запуск приложения
```bash
python main.py
```

## 📖 Примеры использования

### Пример 1: Создание простого напоминания
1. Нажмите **"➕ Добавить"**
2. Заголовок: `Купить молоко`
3. Описание: `Молоко, хлеб, яйца`
4. Время: `2026-06-06 19:00:00`
5. Нажмите **"💾 Сохранить"**

### Пример 2: Быстрое напоминание через 5 минут
1. Нажмите кнопку **"5 мин"**
2. Заголовок: `Проверить почту`
3. Описание: `Важные письма`
4. Нажмите **OK**

### Пример 3: Фильтрация напоминаний
1. Нажмите **"Просрочено"** чтобы увидеть только просроченные
2. Отметьте их как **"✅ Готово"** или **"❌ Отменить"**
3. Нажмите **"Все"** чтобы вернуться к полному списку

## 🎯 Основные функции

| Кнопка | Действие |
|--------|----------|
| ➕ Добавить | Создать новое напоминание |
| 🗑️ Удалить | Удалить выбранное напоминание |
| ✅ Готово | Отметить как выполненное |
| ❌ Отменить | Отменить напоминание |
| 🔄 Обновить | Обновить список |
| 🧪 Тест | Тестовое уведомление |

## 💡 Советы

✨ **Экономия времени:**
- Используйте кнопки **1 мин**, **5 мин**, **15 мин**, **1 час** для быстрого создания напоминаний
- Двойной клик на напоминание показывает полные детали

✨ **Организация:**
- Используйте фильтры для разных типов задач
- Удаляйте завершенные напоминания регулярно

✨ **Уведомления:**
- Приложение проверяет напоминания каждую секунду
- Уведомления работают даже при свернутом окне
- Нажмите **"🧪 Тест"** для проверки уведомлений

## 🔍 Формат даты и времени

Используйте формат: **YYYY-MM-DD HH:MM:SS**

Примеры:
- `2026-06-06 14:30:00` - 6 июня 2026, 14:30
- `2026-12-25 08:00:00` - 25 декабря 2026, 08:00
- `2026-06-08 23:59:59` - 8 июня 2026, 23:59

## 🐛 Частые ошибки

**Ошибка:** "ModuleNotFoundError: No module named 'win10toast'"
- Выполните: `pip install -r requirements.txt`

**Ошибка:** "Неверный формат времени"
- Используйте: `YYYY-MM-DD HH:MM:SS`

**Ошибка:** Уведомления не появляются
- Проверьте: Может ли Windows показывать уведомления?
- Приложение покажет pop-up вместо системного уведомления

## 📊 Состояние базы данных

База данных **reminders.db** хранится в папке приложения.

Таблица `reminders`:
```sql
id          - Уникальный идентификатор
title       - Заголовок напоминания
description - Описание
due_time    - Дата и время срабатывания
status      - Текущий статус
created_at  - Время создания
```
