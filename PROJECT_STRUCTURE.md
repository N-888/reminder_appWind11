## Описание файлов

### 🚀 Основные модули

#### `main.py` - Точка входа
- **Назначение:** Запуск приложения
- **Размер:** ~50 строк
- **Зависимости:** database.py, notifications.py, gui.py
- **Функция:** 
  - Инициализирует БД
  - Запускает менеджер уведомлений
  - Создаёт GUI
  - Запускает главное окно

**Запуск:**
```bash
python main.py
```

---

#### `database.py` - Работа с БД
- **Назначение:** SQLite3 операции с напоминаниями
- **Размер:** ~200 строк
- **Класс:** `ReminderDatabase`
- **Методы:** 20+ для работы с БД
- **Возможности:**
  - CRUD операции (Create, Read, Update, Delete)
  - Фильтрация и сортировка
  - Автоматическая разметка просроченных

**Пример:**
```python
from database import ReminderDatabase
db = ReminderDatabase("reminders.db")
db.add_reminder("Встреча", "Описание", "2026-06-06 14:00:00")
```

---

#### `notifications.py` - Уведомления
- **Назначение:** Система уведомлений в реальном времени
- **Размер:** ~150 строк
- **Класс:** `NotificationManager`
- **Методы:** Мониторинг, уведомления, тестирование
- **Возможности:**
  - Фоновый мониторинг (1 сек интервал)
  - Win10Toast поддержка
  - Fallback на Tkinter Pop-up
  - Звуковое оповещение

**Пример:**
```python
from notifications import NotificationManager
notifier = NotificationManager(db)
notifier.start_monitoring()
```

---

#### `gui.py` - Графический интерфейс
- **Назначение:** Tkinter интерфейс
- **Размер:** ~400 строк
- **Класс:** `ReminderGUI`
- **Виджеты:** Buttons, Listbox, Text, Labels
- **Функции:** 20+ методов для управления напоминаниями
- **Возможности:**
  - Добавление/удаление напоминаний
  - Фильтрация по статусу
  - Просмотр деталей
  - Быстрые кнопки

**Пример:**
```python
import tkinter as tk
from gui import ReminderGUI
root = tk.Tk()
gui = ReminderGUI(root, db, notifier)
gui.run()
```

---

#### `config.py` - Конфигурация
- **Назначение:** Константы и настройки
- **Содержит:**
  - Пути к БД
  - Временные интервалы
  - Цвета и шрифты
  - Допустимые статусы
  - Форматы даты/времени

**Использование:**
```python
from config import DATETIME_FORMAT, COLOR_SUCCESS
```

## Процессы

### Главный поток (Main)
```
main.py
  ├── Инициализация БД
  ├── Инициализация уведомлений
  ├── Создание GUI
  └── Запуск Tkinter (блокирующий вызов)
```

### Фоновый поток (Monitor)
```
NotificationManager
  └── _monitor_loop() [Беззаботный демон]
      ├── Каждую секунду проверяет БД
      ├── Ищет срабатывающие напоминания
      ├── Отмечает просроченные
      └── Показывает уведомления
```

### Поток UI (Tkinter)
```
GUI (Главное окно)
  ├── Кнопки управления
  ├── Список напоминаний
  ├── Фильтры
  └── Диалоговые окна
```

---

## Жизненный цикл приложения

1. **Запуск:** `python main.py`
2. **Инициализация:**
   - Создаёт/открывает БД (reminders.db)
   - Инициализирует менеджер уведомлений
   - Запускает фоновый мониторинг
3. **UI Запуск:**
   - Показывает главное окно
   - Загружает напоминания из БД
   - Отображает в списке
4. **Работа:**
   - Фоновый мониторинг проверяет каждую секунду
   - При срабатывании показывает уведомление
   - Пользователь управляет через UI
5. **Закрытие:**
   - Останавливает мониторинг
   - Закрывает БД
   - Завершает приложение

## Как расширить проект?

### Добавить новую функцию в БД
1. Добавить метод в `database.py` в класс `ReminderDatabase`
2. Использовать `self.cursor.execute()`

### Добавить новую кнопку в GUI
1. Добавить `tk.Button()` в `gui.py` в методе `setup_ui()`
2. Привязать функцию через `command=`

### Добавить новый тип уведомления
1. Добавить метод в `notifications.py`
2. Вызвать из `_show_notification()`

### Добавить конфигурацию
1. Добавить константу в `config.py`
2. Импортировать в нужный модуль
